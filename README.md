# DiffPBR: Point-Based Rendering via Spatial-Aware Residual Diffusion

This repository hosts the project page for **DiffPBR: Point-Based Rendering via Spatial-Aware Residual Diffusion**.

Project page: https://yipingx.github.io/diffpbr.github.io  
Paper: https://openreview.net/pdf?id=tqOBZbW6j8

## Authors

Yiping Xie, Yuchi Huo, Yunlong Ran, Zijian Huang, Lincheng Li, Yingfeng Chen, Jiming Chen, Qi Ye

## Overview

DiffPBR is a diffusion-based framework for synthesizing coherent, photorealistic renderings from diverse point cloud inputs. It uses viewpoint-projected noise constrained by scene geometry and visibility to improve geometric consistency across camera motion, combines adaptive CoNo-Splatting for efficient point-cloud rasterization, and integrates residual learning into the neural re-rendering pipeline.
